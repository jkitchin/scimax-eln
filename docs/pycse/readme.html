1. Solve the PFR several times

1.1. small rate constant

from scipy.integrate import solve_ivp

k = 0.1
nu = 1
Vspan = (0, 2)
Fa0 = 1

def ode(V, Fa, k):
    return -k * Fa / nu

sol = solve_ivp(ode, Vspan, (Fa0,), args=(k,))
sol.y[0][-1]

1.4. run it in a loop

K = [k, 2*k, 4*k]

Fa = [solve_ivp(ode, Vspan, (Fa0,), args=(_k,)).y[0][-1] for _k in K]
